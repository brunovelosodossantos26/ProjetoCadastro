Dynamic version (PHP version) details:

All templates come with static (html) versions of template files along with all assests (css, javascript, images...).

99% of web developers need a dynamic admin theme (PHP), easy to modify. We understand how frustrating and time consuming it can be to transform static HTML theme into a dynamic version (PHP). For Pro and Extended versions, you will have dynamic version also.

This means that all common blocks from your template files (such as navigation menu, common head code, footer code...) will be in separate php files. In this way, you will save a lot of hours of work to sepparate common code into separate files.
